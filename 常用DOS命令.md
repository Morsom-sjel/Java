## 常用DOS命令

**切换盘符**                                      盘符名称：

**进入文件夹**                                  cd 文件夹名称

**进入多级文件夹**                          cd 文件夹1\文件夹2\文件夹3

**返回上一级**                                  cd ..

**直接回根路径**                              cd \

**查看当前文件夹内容**                  dir

**退出**                                             exit
